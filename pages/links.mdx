# Enlaces útiles

* [Obtener ip pública](https://ip.alvarosaavedra.es): obtención de tu ip pública en texto plano.
* [Generador de contraseñas](https://genera-password.gonzaleztroyano.es/) para más información sobre como utilizar esta página visita el blog de [Pablo Gonzalez Troyano](https://blog.gonzaleztroyano.es/cloudflare-workers-password-generator/)
* [Excalidraw](https://excalidraw.com/): crear diseños de arquitectura sencillos y amigables.
* [MXToolbox](https://mxtoolbox.com/): todo lo necesario para buscar DNS records, whois, email servers, etc.
* [QR generator](https://qr.alvarosaavedra.es): creador códigos QR a partir de una URL.
* [It-tools](https://it-tools.tech/): herramientas útiles para desarrolladores, SysAdmin, etc.
* [Chmod help](https://ss64.com/bash/chmod.html) ayuda para usar el comando chmod de linux.
* [Base64 encode](https://b64.alvarosaavedra.es/?action=encode&text=yourText) para codificar texto en base64.
* [Base64 decode](https://b64.alvarosaavedra.es/?action=decode&text=eW91clRleHQ=) para descodificar texto en base64.
