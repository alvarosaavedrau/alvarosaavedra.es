# Cloudflare Workers

## ¿Qué son los Cloudflare Workers?

Los [Cloudflare Workers](https://workers.cloudflare.com) son una forma de ejecutar código en la nube, de manera serverless. Es muy útil para pequeñas funciones que no requieren mucha computación. Si estás familiarizado con AWS, es muy similar a una Lambda.

## Características principales

- **Desempeño:** Los Workers corren en más de 200 centros de datos en todo el mundo, ofreciendo baja latencia.
- **Modelo de facturación:** Paga únicamente por las solicitudes y el tiempo de ejecución.
- **Integración sencilla:** Fácil de implementar y compatible con herramientas estándar como GitHub Actions.
- **Escalabilidad automática:** Se adapta automáticamente a las demandas del tráfico.

## Ventajas frente a otras tecnologías

Comparado con otras soluciones serverless, Cloudflare Workers ofrece:

- **Inicio rápido:** Los Workers se inician en menos de 5ms.
- **Sin administración de infraestructura:** No necesitas preocuparte por servidores o contenedores.
- **Distribución global automática:** El código se ejecuta cerca del usuario final.

## Limitaciones

Es importante considerar las siguientes restricciones:

- **Restricciones de tiempo de ejecución:** Cada Worker tiene un límite de 50ms por ejecución.
- **Límites de tamaño:** El tamaño máximo de un script es de 1MB.
- **Costos para casos de uso masivo:** Puede ser más caro si necesitas procesar grandes cantidades de datos o almacenar información persistente.
